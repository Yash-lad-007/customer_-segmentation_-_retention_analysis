# Customer Segmentation & Retention Analysis

## Overview
This project analyzes customer purchasing behavior using RFM segmentation and cohort retention analysis.

## Tools Used
- Python
- Pandas
- Scikit-learn
- Matplotlib / Seaborn

## Key Results
- Identified 4 customer segments
- Found retention drop after Month 2
- Highlighted high-value customers

## Business Impact
Insights can help target marketing and improve retention.
